# Change Planning

Plan in artifacts, not prose.

## Workflow

1. Define objective and non-goals.
2. Split work into minimal executable steps.
3. Identify dependencies, rollback points, and verification tasks.
4. Output acceptance criteria and risk controls.

## Output Checklist

- change scope
- affected components
- test strategy
- rollback strategy
- acceptance criteria
