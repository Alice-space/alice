# Result Reporting

Report outcomes with traceability.

## Workflow

1. Gather final artifacts and status facts.
2. Summarize decisions, evidence, and unresolved risks.
3. Present user-facing output in concise sections.
4. Include next actions when task is not terminal.

## Guardrails

- Do not omit blocking failures or pending gates.
- Distinguish final result from intermediate progress.
- Keep links/refs to primary evidence.
