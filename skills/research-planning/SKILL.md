# Research Planning

Define measurable experiments before execution.

## Workflow

1. Freeze objective metric and threshold.
2. Specify dataset, baseline, and evaluation protocol.
3. Set budget caps and hard stop conditions.
4. Define success/fail branches for next step routing.

## Output Checklist

- metric and threshold
- dataset and baseline references
- budget and resource caps
- stop criteria
- expected artifacts
